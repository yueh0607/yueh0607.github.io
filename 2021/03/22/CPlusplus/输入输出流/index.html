C++ 输入输出流
数据类型 描述
ofstream 该数据类型表示输出文件流，用于创建文件并向文件写入信息。
ifstream 该数据类型表示输入文件流，用于从文件读取信息。
fstream 该数据类型通常表示文件流，且同时具有 ofstream 和 ifstream 两种功能，这意味着它可以创建文件，向文件写入信息，从文件读取信息。

头文件

#include<fstream>
using namespace std;

fstream

这里只写fstream，ifstream和ostream相差不多，cin和cout也是同理，还有istringstream,ostringstream都是相似的，本质上都是一种流操作器，他们的行为也高度统一。

构造函数

ofstream outfile(string path,ios::openmode mode);  //初始绑定文件
ofstream outfile(ios::openmode mode);   //需要后续使用open打开文件

打开文件

在从文件读取信息或者向文件写入信息之前，必须先打开文件。

void open(const char *filename, ios::openmode mode=ios::out);

在这里，fstream.open() 函数的第一参数指定打开的路径，第二个参数为文件打开模式

模式标志 描述
ios::app 追加模式
ios::ate 文件指针初始指向末尾
ios::in 只读打开
ios::out 只写打开
ios::trunc 如果文件存在，则置空打开
ios::binary 二进制方式

如果希望在打开的同时置空文件，可以按以下写法

ifstream  afile;
afile.open("file.dat", ios::out | ios::in);

关闭文件

当 C++ 程序终止时，一般的操作系统会自动回收所有非托管资源，并且关闭所有文件。但是部分操作系统不能进行这种过程，甚至在程序结束时连内存都无法回收。

如果不刷新缓冲区，可能导致内容写入缓冲区却没有进入文件，我们有必要在使用结束后关闭文件，关闭时将自动刷新缓冲区。

afile.close();

读写方式

afile<<"Aaa"<<endl;

int a;
afile>>a;

filebuf* rdbuf() 返回filebuf指针
ifstream& operator= (const ifstream&) = delete;

istream& read (char* s, streamsize n); 从输入流中提取n个字符，并把他们存数组s中，不检测内容，也不加字符串结尾符号‘\0’

cin/cout进制转换

cout << oct << a << endl; 		//八进制 
cout << dec << a << endl; 		//十进制
cout << hex << a << endl;		//十六进制
	
bitset<8> bs(a);                //转换成二进制
cout << bs << endl; 

//同理，cin也可以指定输入进制---------------------

cout.setf(ios::right | ios::hex); //设置16进制右对齐
cout.setf(ios::right, ios::adjustfield); //取消其它对齐，设置为右对齐
格式符 效果
ios::dec 以10进制表示整数
ios::hex 以16进制表示整数
ios:9oct 以8进制表示整数
ios::showbase 为整数添加一个表示其进制的前缀
ios::internal 在符号位和数值的中间插入需要数量的填充字符以使串两端对齐
ios::left 在串的末尾插入填充字符以使串居左对齐
ios::right 在串的前面插入填充字符以使串居右对齐
。ios::boolalpha 将bool类型的值以true或flase表示，而不是1或0
ios::fixed 将浮点数按照普通定点格式处理（非科学计数法）
格式符 效果
ios::showpoint 在浮点数表示的小数中强制插入小数点（默认情况是浮点数表示的整数不显示小数点）
ios::showpos 强制在正数前添加+号
ios::skipws 忽略前导的空格（主要用于输入流，如cin）
ios::unitbuf 在插入（每次输出）操作后清空缓存
ios::uppercase 强制大写字母
