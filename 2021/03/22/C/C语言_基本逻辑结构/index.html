C语言-基本逻辑结构

1.if-else结构

默认自动结合下一句，有大括号结合大括号

int main()
{
	int a = 10;
	if(a>=10)
	{
		a = 5;
	}
	else if(a>=20)
	{
		a=10;
	}
	else a = 0;
	return 0;
}

2.switch -case结构

其实可以不加大括号的，大括号是作用域规则

int main()
{
	int a = 10;
	switch(a)
	{
		case 1:
		case 2:a = 4;break;
		default:a = 0;break;	
	}
	//如果break缺省，则进入后执行到下一个break;
	//这样写可以让a=1||a=2时进入第二种情况
	return 0;
}

3.while循环

1当表达式为假时（0为假），不执行while语句中的内容。

2当表达式为真的时候（非0），便循环执行while循环语句的内容，直到表达式为假的时候便跳出循环。

3当while循环语句中出现break语句时，break语句的作用是结束所有循环。

4当while循环语句中出现continue语句时，continue语句的作用是结束本次循环。

int main()
{
	int a = 0;
	while(a<=100)
	{
		a++;
	}
	printf("%d",a);


	do
	{
		printf("哈哈");
		a++;
	}
	while(a<200);
	
	return 0;
}

4.for循环

在for 循环体内修改循环变量int i，可能引发计数失控

int main()
{
	for(int i=0;i<10;i++)  //++i也可以，不要纠结，性能几乎没有差异
	{
		printf("第%d次",i);
	}
}

顺序：int i = 0 i<10 code i++ i<10 code i++ ….
