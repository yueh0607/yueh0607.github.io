C语言-输入输出与转义

第一个程序与main函数

stdio.h = standard input output . head

main函数是C语言定义的入口，程序从这里顺次执行，main函数在一个项目里具备唯一性

也就是说一个C程序只有一个main函数

#include <stdio.h>   //通过预编译指令导入头文件，在编译时导入
//#include"stdio.h"   
//这种也是允许的，这种方式将先检索用户自定义的头文件，然后再到系统定义的里寻找，上面的方式正好相反
//通常建议系统头文件均采用第一种方式


//过去常常写void作为返回,新的写法是第二种，其实写float...都可以执行
/*
	void main()
	{
	
	}
*/


//如果返回0,则代表程序正常退出,否则代表程序异常退出
//返回值表示了程序的退出状态，这是建议的一种写法
int main()
{
	//控制台输出函数
	printf("Hello,World!\n");
	return 0;
}

简单输入与输出

#include <stdio.h> 
int main()
{
    int value;
    scanf("value=%d",&value);
	//控制台输出函数
	printf("Hello,World!%d\n",value);
	return 0;
}

通过简单的占位符，我们能进行输入和输出，其中scanf前面的字符串内必须进行相同的输入，但是我们需要用值替换%d,像这样输入

value=30
//如果是scanf("%d",&value);我们只需要输入30即可

另外scanf有溢出风险，因为输入的长度是不确定的，直接赋值给value，有可能导致越界访问，这样可能影响其他程序的安全性，所以在VisualStudio中禁用scanf，采用scanf_s代替。

需要注意scanf必须传入的是地址表列，而不是值。

常用占位符

%d 十进制整数(int)
%ld long
%lld long long
%u unsigned int
%o 八进制整型
%x 十六进制整数/字符串地址
%c 单个字符
%s 字符串
%f float，默认保留6位
%lf double
%e 科学计数法
%g 根据大小自动选取f或e格式，去掉无效0

转义符

转义符可以取消关键字符的特殊性，下面是常见的转义符使搭配

printf("\b");//退格符
printf("\n");//换行
printf("\a");//响铃，电脑真的响一下，不可思议
printf("\t");//水平制表符
printf("\v");//垂直制表符
printf("\130");//输出char对应130的字符
printf("%% %d",12);//%的转义使用%,而不是\
