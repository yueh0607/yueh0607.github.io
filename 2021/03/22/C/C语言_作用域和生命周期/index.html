C语言-作用域和生命周期

搞清楚两个内容

1.生命周期: 局部变量是整个局部结束，全局变量是整个程序

2.作用域

hello.c

int val = 150;

world.c

#include<stdio.h>

int c = 20;  //全局变量生命周期是程序生命周期
extern int val;   //也可以写在函数里作为外部变量，不可在这修改

int main()
{
	int b = 100;  //局部变量生命周期是整个局部结束，这里是函数结束
	{
		int a = 10;  //大括号结束就释放内存
	}
	return 0;
}
